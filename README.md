# Vue-magic

Mon premier TP avec le framework Vue.  
Fonctionne avec axios et donc [APIHero](https://gitlab.com/Timomoulin/APIHero) de mon fromateur.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
